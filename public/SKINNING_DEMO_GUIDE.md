# 🎯 Skinning Data Visualizer - 사용 가이드

## 개요
스키닝 데이터 시각화 도구는 `skin_weight.py`로 생성된 프록시 노드와 스킨 가중치를 인터랙티브하게 확인할 수 있는 웹 기반 3D 뷰어입니다.

## 설치 및 실행

### 1. 스키닝 데이터 생성
먼저 Python 도구로 스킨 가중치 데이터를 생성합니다:

```bash
cd /home/ubuntu/nerfstudio
python skin_weight.py input.ply -n 24 --sigma 5.0
# 출력: input_nodes24_sigma5.0/proxy_nodes.json + skinning_data.bin
```

### 2. 웹 서버 시작
```bash
# 프로토타입 디렉토리로 이동
cd /home/ubuntu/nerfstudio/prototype/public

# Python 내장 웹 서버 시작 (포트 8000)
python -m http.server 8000

# 또는 Node.js를 사용하는 경우:
# npx http-server -p 8000
```

### 3. 브라우저에서 열기
```
http://localhost:8000/skinning_demo.html
```

## 사용 방법

### 데이터 로드

1. **데이터 경로 입력**
   - HTML 페이지의 "데이터 경로" 입력 필드에 생성된 폴더 경로 입력
   - 예: `../path/to/input_nodes24_sigma5.0`

2. **🔄 데이터 로드 버튼 클릭**
   - proxy_nodes.json과 skinning_data.bin이 로드됨
   - 상태창에 로드된 노드 개수 표시
   - 프록시 노드가 파란색 구 형태로 씬에 표시됨

### 시각화 옵션

| 옵션 | 설명 |
|------|------|
| ✅ 가우시안 스플랫 표시 | GS 모델 렌더링 (향후 지원) |
| ✅ 스키닝 포인트 표시 | 프록시 노드 구체 표시 |
| ☐ 본 연결선 표시 | 인접한 노드들 간 선 연결 |

### 본 제어

1. **본 선택**
   - 슬라이더로 0~N-1 범위의 본 선택
   - 선택된 본은 **빨간색**으로 표시
   - 다른 본들은 **파란색**으로 표시

2. **회전 조정**
   - **X 회전**: -180° ~ +180°
   - **Y 회전**: -180° ~ +180°
   - **Z 회전**: -180° ~ +180°
   - 각 슬라이더 조정 시 본의 4×4 변환 행렬이 업데이트됨

3. **카메라 제어**
   - **마우스 드래그**: 씬 회전
   - **마우스 휠**: 줌 인/아웃
   - **카메라 거리**: 슬라이더로 수동 조정 (10~500)

## 기술 상세

### 데이터 형식

**proxy_nodes.json**
```json
[
  { "x": 0.5, "y": 10.0, "z": 0.2 },
  { "x": 1.2, "y": 11.5, "z": 0.1 },
  ...
]
```

**skinning_data.bin**
- Float32 바이너리 형식
- 각 포인트당 K개의 top weights 저장
- 레이아웃: [point_0_indices... point_0_weights... point_1_indices... ...]

### 본 행렬 저장
- 각 본의 4×4 변환 행렬을 Three.js DataTexture에 저장
- Column-major 순서로 저장 (WebGL 호환)
- NearestFilter 사용하여 정확한 텍스처 읽기 보장

### 상호작용 루프
1. 슬라이더 조정 → `updateBoneRotation()` 호출
2. 회전 오일러 각도를 3×3 회전 행렬로 변환
3. 4×4 행렬로 확장 (좌상단에 회전, 우측/아래 아이덴티티)
4. 텍스처 데이터 업데이트 및 GPU 동기화
5. 다음 프레임에 렌더러가 업데이트된 행렬을 사용

## 문제 해결

### "데이터 로드 실패: 404"
- 데이터 경로 확인
- 웹 서버의 기준 디렉토리 확인
- 상대 경로가 올바른지 확인

### 크로스 오리진 에러 (CORS)
- 같은 웹 서버에서 실행 필요
- 로컬 파일:// 프로토콜 지원 안 함

### 노드가 보이지 않음
- "스키닝 포인트 표시" 체크박스 확인
- 카메라 거리 조정
- 마우스로 씬 회전

### 회전이 반영되지 않음
- 콘솔 로그에서 "회전 업데이트" 메시지 확인
- 본이 선택되었는지 확인 (빨간색)
- 슬라이더를 천천히 조정하며 변화 관찰
